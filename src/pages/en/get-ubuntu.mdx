# Commercial support

> **Note**
> The Ubuntu Korean community is an independent user and developer community operated as a non-profit organization, separate from Canonical Ltd. It does not provide commercial support for Ubuntu, nor can it respond to related inquiries. If needed, please refer to the information below to receive commercial support from Canonical Ltd. or Canonical Partners in Korea or abroad.

## Ubuntu Pro
You can receive commercial support from Canonical Ltd., the developer of Ubuntu, through Ubuntu Pro.  
There are free, Infra-only, and full support plans available, providing technical support for desktop, server, VM, and applications (MySQL, MongoDB, NGINX, Kafka, etc.).

[Ubuntu Pro (English)](https://ubuntu.com/pro)

## Ubuntu Pro for public cloud
When using AWS, Google Cloud, or Microsoft Azure, you can create VM workloads with Canonical Ltd.'s paid premium Ubuntu Pro images and receive commercial support without additional contracts.

- [Ubuntu Pro for AWS (English)](https://ubuntu.com/aws/pro)
- [Ubuntu Pro for Azure (English)](https://ubuntu.com/azure/pro)
- [Ubuntu Pro for Google Cloud (English)](https://ubuntu.com/gcp/pro)

## Canonical Partner
Commercial support is also provided by Canonical Ltd.'s partners.

[Find a Canonical partner (English)](https://canonical.com/partners/find-a-partner)
