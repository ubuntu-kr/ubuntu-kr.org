## Bridge services between platforms
To link channels across platforms we use the bridge services below. Delayed or missing messages between platforms can happen if there is an issue with these bridges; check their status for more details.

- t2bot.io: Discord–Matrix bridge: [About](https://t2bot.io/), [Status](https://status.t2bot.io/785547346)
- Libera.chat IRC Bridge: Matrix–IRC bridge: [About](https://matrix-org.github.io/matrix-appservice-irc/latest/bridged_networks.html), [Status](https://status.matrix.org/)

## IRC connection info

If you use an IRC client, connect with the details below.

- Server: irc.libera.chat (or irc.ubuntu.com, which points to irc.libera.chat)
- Ports
  - Plain: `6665–6667` or `8000–8002`
  - TLS: 6697, 7000, 7070
- Channel: #ubuntu-kr
- [More about using IRC](https://wiki.ubuntu-kr.org/index.php/IRC_%EC%9D%B4%EC%A9%A9_%EC%A0%95%EB%B3%B4)

Before joining IRC, please review the rules below:
- [IRC rules](https://wiki.ubuntu.com/KoreanTeam/IRC_Rules)
- [Ubuntu IRC channel Terms of Service](https://wiki.ubuntu.com/IRC/TermsOfService)
